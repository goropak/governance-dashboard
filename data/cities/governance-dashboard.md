## 창설 배경

거버넌스 국가의 모든 도시를 한눈에 파악하고, 시대별 흐름을 문명사적 시각으로 조망하기 위해 만든 포털. 도시 목록을 텍스트로만 관리하던 한계를 벗어나 시각적 지도로 국가 전체를 탐색할 수 있게 한다.

## 핵심 기능

- **문명사 포털** — 4개 시대(인지·농업·산업·정보혁명기)로 국가 역사를 조망
- **SVG 추상 문명 지도** — 9개 도시를 시대별 영역에 배치, 마커 클릭으로 상세 확인
- **시대 타임라인** — 상단 가로 바에 시대별 도시 창설 시점 표시
- **소통 가이드** — 보좌관↔시장 협업 방식 문서 열람
- **대통령 Vault 안내** — 참조 레이어 사용설명서

## 기술 스택

- 순수 HTML / CSS / JS (빌드 도구 없음)
- ES Module 구조 (sync.js → 중앙 상태, data.js → JSON fetch)
- Google Fonts (Noto Serif KR, Pretendard)
- Vercel 정적 배포

## 사용 방법

포털을 열면 자동으로 데이터를 로드한다. 지도의 도시 마커를 클릭하거나 타임라인의 점을 클릭하면 우측 패널에 도시 상세 정보가 표시된다.
