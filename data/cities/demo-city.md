## 창설 배경

보좌관↔시장 메일박스 통신과 telegram-gate 승인 루프가 실제로 end-to-end 작동하는지 격리된 환경에서 실연하기 위해 만든 데모 도시. 실 데이터·실 코드 없이 루프 구조 자체를 검증하는 샌드박스다.

## 핵심 기능

- outbox → 시장 지시 → 처리 → inbox 보고 → 대통령 확인 루프 실연
- telegram-gate 승인 흐름 격리 테스트
- 거버넌스 협업 패턴의 살아있는 문서

## 기술 스택

- Shell (표준 Unix 툴)
- 텍스트 파일 기반 메일박스

## 사용 방법

데모 실연 전용. 일반 사용 시나리오 없음. 필요 시 보좌관이 지시서 작성 → 시장이 처리 → inbox 보고 순서로 루프 시연.

## 연결

- GitHub: 미생성 (로컬 전용)
- 관련 도시: telegram-gate (승인 루프 실연 파트너)
