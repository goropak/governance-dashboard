# 도시 명부

운영 중인 모든 프로젝트(=도시)의 인덱스.
각 도시의 상세는 해당 레포의 `STATUS.md`와 `INFRA.md` 참조.

## 형식 예시

### [도시 이름]
- **Repo**: github.com/.../...
- **Deploy**: ...
- **DB**: ...
- **Status**: 운영중 / 개발중 / 방치 / 폐쇄
- **Last check**: YYYY-MM-DD
